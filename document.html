WEEK 3 : INTRODUCTION TO CSS

CSS (Cascading Style Sheets) is a fundamental technology used to style and enhance the appearance of web pages. It allows developers to control colors, fonts, layouts, and overall design, making websites more visually appealing and user-friendly. CSS works by applying styles to HTML elements using different selectors, such as element, class, and ID selectors
